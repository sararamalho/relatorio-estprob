📌 **Passos para utilização do Projeto relatorio-estprob**

1. Baixe o projeto compactado clicando no botão [`<> Code > Local > Download ZIP`](https://github.com/bendeivide/relatorio-estprob/archive/refs/heads/master.zip);
2. Tenha instalado o R, RStudio e Git, acesse o [link](https://bendeivide.github.io/relatorio-estprob/relatorio/index.html#vis%C3%A3o-geral-sobre-os-relat%C3%B3rios);
3. Depois em [3 Visão geral sobre os relatórios](https://bendeivide.github.io/relatorio-estprob/relatorio/index.html#vis%C3%A3o-geral-sobre-os-relat%C3%B3rios), siga os passos: 3.1 a 3.5.
