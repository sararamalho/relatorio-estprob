2 📚 Fundamentação Teórica

A linguagem R é um ambiente de programação utilizado para análise de dados e estatística. Seu funcionamento baseia-se na execução de comandos que criam e manipulam objetos armazenados na memória. Os dados podem ser organizados por meio de estruturas como vetores, matrizes, data frames e listas, enquanto scripts e arquivos de sessão auxiliam na organização e reprodutibilidade das análises.

2.1 A seguir , os passos iniciais para utilização do R e RStudio:

Instalação: Baixe o R no site oficial (CRAN). Instale o RStudio (interface gráfica que facilita o uso do R). Abrindo o RStudio: Você verá \(4\) áreas principais: Console: onde os comandos são executados; Script: onde você escreve códigos; Environment: mostra objetos criados; Files/Plots/Packages: arquivos, gráficos e pacotes.

Primeiro comando: 2 + 2

Criando um script
Clique em File → New File → R Script

Escreva: x <- 10 y <- 5 x + y Execute com Ctrl + Enter

2.2 História do R

O R foi criado na década de 1990 por Ross Ihaka e Robert Gentleman, baseado na linguagem S. Seu objetivo era fornecer uma ferramenta livre para análise estatística. Hoje, é amplamente utilizado em ciência de dados, pesquisa e engenharia.

Como o R trabalha?

O R funciona como um interpretador:

executa comandos linha por linha; armazena resultados em objetos; permite testar e corrigir rapidamente.

Exemplo:

\(a\) <- \(5\)

\(b\) <- \(2\)

\(a * b\)

resultado : \(10\)

Comandos no R e Ambiente Global

Comandos: instruções dadas ao R

Ambiente Global: onde ficam os objetos criados

Exemplo:

nota <- 8.5

👉 O objeto nota aparecerá no Environment

2.3 Arquivos .RData e .Rhistory

.RData: salva todos os objetos criados .Rhistory: salva o histórico de comandos

Salvar:

save.image() # salva .RData savehistory() # salva histórico

Carregar:

load(“arquivo.RData”)

2.4 Criando e salvando um Script

Script = arquivo .R com comandos

Permite:

organização; reutilização; reprodutibilidade.

Salvar:

File → Save As → nome.R

2.5 Objetos e estruturas de dados

Um objeto é um local na memória do computador onde o R guarda uma informação. No R, os dados são armazenados em objetos. Existem diferentes estruturas para organizar essas informações, cada uma adequada a um determinado tipo de análise.

🔹 Exemplos:

Vetor

O vetor é a estrutura mais simples do R. Ele armazena vários valores de um mesmo tipo (números, textos ou valores lógicos) em uma única sequência.

Código 
x <- c(1, 2, 3, 4)

x
[1] 1 2 3 4

c() significa “combinar”; cria uma sequência de valores do mesmo tipo.

Matriz

tabela numérica organizada em linhas e colunas

Código 
m <- matrix(1:6, nrow = 2)

m
     [,1] [,2] [,3]
[1,]    1    3    5
[2,]    2    4    6

1:6 gera os números de 1 até 6; nrow = 2 cria 2 linhas; o R distribui os valores por colunas.

Data Frame

tabela semelhante ao Excel, com colunas que podem ter tipos diferentes.

Código 
dados <- data.frame(
  nome = c("Ana", "João"),
  idade = c(20, 22)
)

dados
  nome idade
1  Ana    20
2 João    22

data.frame() cria uma tabela; cada coluna recebe um nome; as colunas podem ter tipos diferentes.

Lista

coleção de objetos de qualquer tipo dentro de uma mesma estrutura.

Código 
lista <- list(
  a = 1,
  b = "texto"
)

lista
$a
[1] 1

$b
[1] "texto"

Uma forma intuitiva de pensar é:

  • Vetor → uma fila de dados.
  • Matriz → uma grade de números.
  • Data Frame → uma planilha.
  • Lista → uma pasta que pode guardar vários tipos de arquivos.
