1 Estimadores Intervalares para Populações Normais e Bernoulli

Introdução

A estimação intervalar é uma das principais ferramentas da Inferência Estatística. Seu objetivo é estimar parâmetros populacionais por meio de intervalos de confiança, fornecendo não apenas uma estimativa do parâmetro, mas também uma medida da incerteza associada ao processo de amostragem.

Neste relatório são estudados intervalos de confiança para médias, variâncias, diferenças entre médias, razão entre variâncias e proporções populacionais. Além disso, são apresentadas implementações computacionais em R para ilustrar a aplicação prática dos métodos estudados.

Objetivo Geral

Estudar a construção e aplicação de estimadores intervalares para parâmetros de populações normais e Bernoulli.

Objetivos Específicos

  • Deduzir o intervalo de confiança para a média populacional com variância conhecida.
  • Deduzir o intervalo de confiança para a variância populacional.
  • Deduzir intervalos para a diferença entre médias de duas populações normais independentes.
  • Deduzir intervalos para a razão entre variâncias populacionais.
  • Estudar intervalos aproximados e exatos para o parâmetro (p) de uma população Bernoulli.
  • Implementar os procedimentos em R.
  • Relacionar os resultados com os conteúdos estudados anteriormente.
  • Apresentar aplicações práticas dos estimadores intervalares.

Fundamentação Teórica

Intervalo para a Média Populacional ((^2) conhecida)

Considere uma amostra aleatória proveniente de uma população normal:

\[ X_1,\ldots,X_n \sim N(\mu,\sigma^2) \]

com variância populacional conhecida.

A média amostral possui distribuição:

\[ \bar X \sim N\left(\mu,\frac{\sigma^2}{n}\right) \]

Padronizando:

\[ Z= \frac{\bar X-\mu} {\sigma/\sqrt n} \]

tem-se:

\[ Z \sim N(0,1) \]

Logo, o intervalo de confiança para a média populacional é:

\[ IC(\mu)= \bar X \pm z_{\alpha/2} \frac{\sigma}{\sqrt n} \]

Intervalo para a Variância Populacional

Se a população é normal, então:

\[ \frac{(n-1)S^2}{\sigma^2} \sim \chi^2_{n-1} \]

Assim, o intervalo de confiança para a variância populacional é:

\[ IC(\sigma^2)= \left[ \frac{(n-1)S^2} {\chi^2_{1-\alpha/2}}, \frac{(n-1)S^2} {\chi^2_{\alpha/2}} \right] \]

Diferença entre Médias Populacionais (Homocedasticidade)

Considere duas populações normais independentes:

\[ X_1,\ldots,X_{n_1} \]

e

\[ Y_1,\ldots,Y_{n_2} \]

assumindo:

\[ \sigma_1^2=\sigma_2^2 \]

A variância combinada é:

\[ S_p^2= \frac{ (n_1-1)S_1^2+ (n_2-1)S_2^2 } {n_1+n_2-2} \]

O intervalo de confiança para a diferença entre médias é:

\[ (\bar X-\bar Y) \pm t_{\alpha/2} S_p \sqrt{ \frac1{n_1} + \frac1{n_2} } \]

Diferença entre Médias Populacionais (Heterocedasticidade)

Quando as variâncias não são iguais:

\[ \sigma_1^2 \neq \sigma_2^2 \]

utiliza-se o método de Welch:

\[ (\bar X-\bar Y) \pm t_{\alpha/2} \sqrt{ \frac{S_1^2}{n_1} + \frac{S_2^2}{n_2} } \]

Razão entre Variâncias Populacionais

Sabendo que:

\[ \frac{ S_1^2/\sigma_1^2 } { S_2^2/\sigma_2^2 } \sim F \]

o intervalo para a razão entre variâncias é:

\[ IC \left( \frac{\sigma_1^2} {\sigma_2^2} \right) = \left[ \frac{S_1^2}{S_2^2} \frac1{F_{1-\alpha/2}}, \frac{S_1^2}{S_2^2} \frac1{F_{\alpha/2}} \right] \]

População Bernoulli

Considere:

\[ X \sim Bernoulli(p) \]

onde:

\[ P(X=1)=p \]

e

\[ P(X=0)=1-p \]

O estimador pontual de (p) é:

\[ \hat p= \frac{x}{n} \]

onde (x) representa o número de sucessos observados.

Intervalo Aproximado para (p)

Pelo Teorema Central do Limite:

\[ \hat p \approx N \left( p, \frac{p(1-p)}{n} \right) \]

resultando no intervalo:

\[ IC(p)= \hat p \pm z_{\alpha/2} \sqrt{ \frac{ \hat p(1-\hat p) } {n} } \]

Fundamento do Intervalo Aproximado

O intervalo aproximado é fundamentado no Teorema Central do Limite, segundo o qual a distribuição amostral da proporção tende à normalidade quando o tamanho da amostra é suficientemente grande.

Intervalo Exato para (p)

Uma alternativa é o método de Clopper-Pearson, baseado diretamente na distribuição binomial.

Esse método produz intervalos exatos, sendo especialmente útil quando o tamanho amostral é pequeno ou quando a proporção está próxima de 0 ou 1.

Relação com o Relatório 6

Os resultados deste relatório estão diretamente relacionados às distribuições Qui-Quadrado e F de Fisher estudadas anteriormente.

A distribuição Qui-Quadrado é utilizada na construção do intervalo de confiança para a variância populacional, pois:

\[ \frac{(n-1)S^2}{\sigma^2} \sim \chi^2_{n-1} \]

Já a distribuição F de Fisher é utilizada na construção do intervalo para a razão entre duas variâncias:

\[ \frac{ S_1^2/\sigma_1^2 } { S_2^2/\sigma_2^2 } \sim F \]

Portanto, os conceitos estudados no relatório anterior constituem a base teórica necessária para a construção dos estimadores intervalares apresentados neste trabalho.

Implementação em R

Intervalo para a Média Populacional

O código abaixo simula uma amostra normal, calcula a média amostral e constrói um intervalo de confiança de 95% para a média populacional.

Código 
set.seed(123)

x <- rnorm(30, mean = 100, sd = 15)

media <- mean(x)
sigma <- 15
n <- length(x)

erro <- qnorm(0.975) * sigma / sqrt(n)

IC <- c(media - erro, media + erro)

cat("Média amostral =", round(media,2), "\n")
Média amostral = 99.29
Código 
cat("IC de 95% para a média = [",
    round(IC[1],2), ",",
    round(IC[2],2), "]")
IC de 95% para a média = [ 93.93 , 104.66 ]

Intervalo para a Variância Populacional

O código calcula a variância amostral e utiliza a distribuição Qui-Quadrado para obter um intervalo de confiança para a variância populacional.

Código 
s2 <- var(x)
n <- length(x)

LI <- (n-1)*s2/qchisq(0.975,n-1)
LS <- (n-1)*s2/qchisq(0.025,n-1)

cat("Variância amostral =", round(s2,2), "\n")
Variância amostral = 216.54
Código 
cat("IC de 95% para a variância = [",
    round(LI,2), ",",
    round(LS,2), "]")
IC de 95% para a variância = [ 137.35 , 391.34 ]

Diferença entre Médias (Variâncias Iguais)

São simuladas duas populações normais independentes e calculado o intervalo para a diferença entre médias.

Código 
x <- rnorm(20,100,10)
y <- rnorm(20,105,10)

resultado <- t.test(x,y,var.equal=TRUE)

resultado$conf.int
[1] -10.6658760   0.3514693
attr(,"conf.level")
[1] 0.95

O intervalo obtido representa valores plausíveis para a diferença entre as médias populacionais.

Diferença entre Médias (Variâncias Diferentes)

Quando não se assume igualdade entre as variâncias:

Código 
resultado <- t.test(x,y,var.equal=FALSE)

resultado$conf.int
[1] -10.6660803   0.3516736
attr(,"conf.level")
[1] 0.95

Neste caso é utilizado o procedimento de Welch.

Razão entre Variâncias

O código abaixo calcula um intervalo para a razão entre as variâncias das duas populações.

Código 
resultado <- var.test(x,y)

resultado$conf.int
[1] 0.3700758 2.3621768
attr(,"conf.level")
[1] 0.95

A construção desse intervalo utiliza a distribuição F de Fisher.

Estimativa para uma Proporção Bernoulli

O exemplo simula uma população Bernoulli com probabilidade de sucesso igual a 0,7.

Código 
dados <- rbinom(100,1,0.7)

sucessos <- sum(dados)

prop.test(sucessos,100)

    1-sample proportions test with continuity correction

data:  sucessos out of 100, null probability 0.5
X-squared = 28.09, df = 1, p-value = 1.158e-07
alternative hypothesis: true p is not equal to 0.5
95 percent confidence interval:
 0.673059 0.845785
sample estimates:
   p 
0.77

O resultado fornece uma estimativa para (p) e seu respectivo intervalo de confiança.

Resultados e Discussões

Os resultados mostram que os intervalos de confiança fornecem uma faixa de valores plausíveis para os parâmetros populacionais.

Observa-se que amostras maiores produzem intervalos mais estreitos, indicando maior precisão nas estimativas. Os intervalos para variância dependem diretamente da distribuição Qui-Quadrado, enquanto os intervalos para razão entre variâncias dependem da distribuição F de Fisher.

Nos casos envolvendo duas populações, a escolha entre o método clássico e o método de Welch depende da hipótese de igualdade ou desigualdade entre as variâncias.

Para populações Bernoulli, o Teorema Central do Limite permite construir intervalos aproximados para a proporção populacional, enquanto o método de Clopper-Pearson fornece uma alternativa exata.

Conclusão

Os estimadores intervalares constituem ferramentas fundamentais da Inferência Estatística, permitindo estimar parâmetros populacionais com uma medida explícita de precisão.

A construção desses intervalos depende das distribuições Normal, Qui-Quadrado, t de Student, F de Fisher e Binomial, evidenciando a importância dos conceitos estudados ao longo da disciplina para a análise e interpretação de dados reais.

**Implementação em R**
